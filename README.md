# MTG Tabletop

一个面向娱乐对局的《万智牌》双人桌面模拟器。

MTG Tabletop 的目标不是实现完整规则引擎，而是提供一张“够顺手、够直觉、适合朋友远程对战”的数字牌桌：玩家自己负责规则判断，工具负责牌库、区域、公开信息、阶段、日志和远程连接。

> 当前版本定位：Tabletop Simulator 风格的轻量牌桌，而非 MTG Arena / MTGO 式自动规则客户端。

---

## 核心特性

### 双人在线牌桌

- 创建 / 加入双人房间。
- 支持本机双开测试、局域网访问和公网临时远程。
- 对局记录同步展示，方便回看关键操作。
- 聊天内容进入对局记录，可作为对局备注使用。

### 牌库与手牌管理

- 支持文本牌表导入，例如：

  ```txt
  4 Lightning Bolt
  4 Counterspell
  24 Island
  ```

- 洗牌、抓牌、抓 7。
- 调度：手牌洗回牌库后抓 7，并记录调度次数。
- 重开：恢复最近导入的牌表，清空公共区域，生命重置为 20。
- 找牌 / 搜牌库：查看自己的牌库并按牌名搜索。
- 看牌库顶 X 张：从牌库顶取出指定数量牌逐张处理，处理一张即减少一张。

### 区域与移动

支持以下区域：

| 区域 | 可见性 | 说明 |
| --- | --- | --- |
| 牌库 | 仅自己 | 支持搜索、找牌、看顶、回顶、回底、洗回 |
| 手牌 | 仅自己 | 可拖动或通过按钮移动 |
| 战场 | 公开 | 分为地与非地区域 |
| 坟场 | 公开 | 双方均可查看详情 |
| 放逐区 | 公开 | 双方均可查看详情 |
| 堆叠 | 公开 | 位于牌桌中央，适合展示咒语和异能 |

牌可以通过两种方式移动：

- 点击选中牌，然后使用左侧“移动选中牌”按钮。
- 直接拖拽到战场、堆叠、坟场、放逐等区域。

回牌库支持：

- 回牌库顶
- 回牌库底
- 洗回牌库

### 真实牌垫风格战场

- 双方对坐布局：对手在上方，自己在下方。
- 对手牌会倒置显示，更接近真实桌面对局视角。
- 地与非地分层摆放：
  - 双方非地靠近中线。
  - 地在各自后排。
- 不自动识别地牌；玩家拖到“地”区域即按地摆放，拖到“非地”区域即按非地摆放。
- 牌面不显示“地 / 生物 / 法术”等类型标签，保持桌面干净。

### 堆叠展示

- 中央公开堆叠区。
- 可将牌拖入堆叠，表示施放咒语或异能上堆叠。
- 堆叠顶部可快速：
  - 进坟场
  - 放逐
  - 进战场
- 坟场 / 放逐区详情中可以把牌移动到堆叠，适合返照等需要从公开区域施放的场景。

### 对局辅助

- 生命值调整。
- 阶段推进：
  - 上一阶段 / 下一阶段。
  - 直接点击阶段编号跳转。
  - 回合结束后自动进入对手维持阶段并重置对应玩家战场。
- 右键战场牌可横置 / 重置。
- +1/+1 指示物。
- 通用计数器。
- 自定义 Token：
  - 可设置名称。
  - 可设置攻防。
  - 可创建无攻防 Token。
  - 自动记忆上一次配置。
- 公开投骰：
  - D2 / D4 / D6 / D8 / D10 / D12 / D20 / D100。
  - 自定义面数。
  - 2D6。
  - 结果写入对局记录。

## 快速开始

### 使用项目脚本启动

如果系统没有安装 Node.js，也可以直接使用项目脚本。脚本会优先使用 Codex 桌面自带的 Node / pnpm。

```powershell
cd D:\MTG-Tabletop
.\scripts\dev.cmd
```

启动后访问：

- 前端：http://127.0.0.1:5180
- 后端健康检查：http://127.0.0.1:8787/health

### 使用 pnpm 启动

如果本机已经配置好 Node.js / pnpm：

```powershell
cd D:\MTG-Tabletop
pnpm install
pnpm dev
```

### PowerShell 脚本方式

如果你偏好 PowerShell 脚本，可以使用：

```powershell
powershell -ExecutionPolicy Bypass -File D:\MTG-Tabletop\scripts\dev.ps1
```

## 局域网访问

同一 Wi-Fi / 局域网下的另一台设备可以访问你的本机服务。

```powershell
cd D:\MTG-Tabletop
.\scripts\show-lan-address.cmd
```

把脚本输出的地址发给同一局域网内的朋友，例如：

```txt
http://你的IPv4:5180
```

## 公网临时远程

项目内置 Cloudflare Quick Tunnel 启动脚本，可生成一个临时公网 HTTPS 地址，适合朋友之间快速远程娱乐对局。

```powershell
cd D:\MTG-Tabletop
.\scripts\remote.cmd
```

脚本会自动完成：

1. 下载 `cloudflared` 到 `D:\MTG-Tabletop\tools\cloudflared.exe`。
2. 构建前端。
3. 启动本地服务 `http://127.0.0.1:8787`。
4. 创建 `https://*.trycloudflare.com` 临时公网地址。

把终端里出现的 `https://*.trycloudflare.com` 链接发给对方即可。

注意事项：

- 游玩期间不要关闭运行窗口。
- Quick Tunnel 是开发 / 测试用途，没有稳定在线保证。
- 每次生成的公网地址可能不同。

## 基本玩法流程

1. 一名玩家创建房间。
2. 将房间码或公网链接发给对手。
3. 双方分别导入牌表。
4. 洗牌。
5. 抓 7。
6. 根据需要调度。
7. 使用阶段按钮推进流程。
8. 通过拖拽或按钮移动牌。
9. 使用堆叠、坟场、放逐、Token、骰子、计数器等辅助工具完成对局。

## 操作说明

### 移动牌

- 点击一张牌后，左侧会显示当前选中牌。
- 使用“移动选中牌”区域可以把牌移动到：
  - 非地战场
  - 地区域
  - 堆叠
  - 坟场
  - 放逐
  - 手牌
  - 牌库顶
  - 牌库底
  - 洗回牌库

### 战场

- 拖到“地”区域，即按地摆放。
- 拖到“非地”区域，即按非地摆放。
- 右键战场上的牌可横置 / 重置。

### 坟场与放逐

- 右侧公开区域会显示双方坟场与放逐区摘要。
- 点击摘要可以打开完整列表。
- 列表中的牌可以移动到：
  - 战场
  - 手牌
  - 堆叠
  - 牌库顶
  - 牌库底
  - 洗回牌库

### 看牌库顶

- 输入数量 X。
- 点击“看牌库顶”。
- 弹窗会显示本次查看的牌。
- 每处理一张牌，弹窗中的牌数会减少。
- 如果关闭弹窗后仍有未处理牌，可以点击“继续处理 N”继续处理。

### Token

- 输入 Token 名称。
- 勾选“有攻防”后可填写攻防。
- 取消勾选可创建无攻防 Token。
- 点击“释放 Token”创建。
- 浏览器会记忆上一次 Token 设置。

## 项目边界

当前版本专注“娱乐桌面模拟”，暂不包含：

- 官方卡图与 Oracle 文本。
- 摩登禁牌表。
- 套牌合法性校验。
- 自动费用支付。
- 自动优先权系统。
- 完整规则引擎。
- 自动裁定。
- 账号系统。
- 匹配 / 天梯。
- 对局回放。
- 服务端持久化。

这些能力可以在后续版本中逐步扩展，但不属于当前 MVP 的核心目标。
