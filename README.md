# MTG Tabletop

一个“可供娱乐”的万智牌双人桌游模拟器 MVP。它不会自动判定完整万智牌规则，而是提供类似桌游模拟器的基础牌桌能力：

- 创建/加入双人房间
- 文本牌表导入
- 洗牌、抓牌
- 调度、重开
- 按规则阶段顺序进入上一阶段/下一阶段、回合结束
- 聊天内容写入对局记录
- 中央公开堆叠区，可拖牌进入堆叠并结算顶部
- +1/+1 指示物和通用计数器
- 自定义释放 Token，支持名称和可选攻防，并记忆上一次配置
- 公开投骰子，结果写入对局记录
- 双方坟场和放逐公开显示摘要，可点开查看详情
- 找地/搜自己的牌库
- 手牌、战场、坟场、放逐、堆叠等区域
- 选牌移动区域，或拖拽到战场/坟场/放逐/堆叠
- 战场是双人对坐牌垫布局：对手在上方，自己在下方
- 双方非地靠近中线，地在各自后排；拖到哪层就按哪层放置
- 对手战场牌会倒置显示，更接近真实桌面对坐感
- 横置/重置
- 右键战场上的牌可横置/重置
- 回牌库支持回顶、回底、洗回去
- 生命值调整
- 对局记录

## 开发运行

本机如果没有安装 Node，可以使用 Codex 桌面自带的 Node/pnpm。

```powershell
cd D:\MTG-Tabletop
pnpm install
pnpm dev
```

如果系统 PATH 里没有 Node/npm，直接用项目自带脚本：

```powershell
cd D:\MTG-Tabletop
.\scripts\dev.cmd
```

也可以双击 `scripts\dev.cmd`。如果你偏好 PowerShell 脚本，需要先允许本次运行策略，例如：

```powershell
powershell -ExecutionPolicy Bypass -File D:\MTG-Tabletop\scripts\dev.ps1
```

然后打开：

- 前端本机：http://127.0.0.1:5180
- 后端健康检查：http://127.0.0.1:8787/health

局域网内其他设备访问：

```powershell
cd D:\MTG-Tabletop
.\scripts\show-lan-address.cmd
```

把脚本显示的 `http://你的IPv4:5180` 发给同一 Wi-Fi/局域网下的另一台设备即可。

## 公网临时远程

公网远程使用 Cloudflare Quick Tunnel。它会生成一个临时 HTTPS 地址，适合开发测试和朋友娱乐对战。

```powershell
cd D:\MTG-Tabletop
.\scripts\remote.cmd
```

脚本会：

1. 自动下载 `cloudflared` 到 `D:\MTG-Tabletop\tools\cloudflared.exe`。
2. 构建前端。
3. 启动本地服务 `http://127.0.0.1:8787`。
4. 创建一个公网 `https://*.trycloudflare.com` 地址。

把终端里出现的 `https://*.trycloudflare.com` 链接发给对方即可。游玩期间不要关闭这个窗口。

注意：Quick Tunnel 是测试/开发用途，临时地址每次可能不同，不适合作为长期稳定线上服务。

## 测试玩法

1. 开一个浏览器窗口，输入玩家名，创建房间。
2. 再开一个浏览器窗口，输入同一个房间码加入。
3. 双方分别导入牌表，洗牌，抓 7。
4. 点击一张牌，再用“移动选中牌”按钮移动到战场/坟场/放逐等区域；也可以直接拖拽。
5. “找地/搜牌”会显示自己的牌库，可以搜索牌名，并把牌移到手牌或战场。
6. “调度”会把当前手牌洗回牌库、抓 7，并记录调度次数；需要玩家手动把对应数量的牌放回牌库。
7. “重开”会清空公共区域、恢复最近导入的牌表、双方生命回到 20。
8. “上一阶段 / 下一阶段”会按规则阶段顺序推进，也可以直接点阶段数字跳转。
9. “回合结束”会进入对手维持阶段并自动重置对手战场；没有对手时会视为对手立即结束，回到自己的维持阶段并重置自己。
10. 右侧聊天框发送的内容会显示在对局记录里。
11. “投骰子”会打开公开骰子工具，支持 D2/D4/D6/D8/D10/D12/D20/D100 和自定义面数。
12. 牌可以拖到中央堆叠；堆叠顶部可以一键进坟、放逐或进场。
13. 右侧公开区域会分别显示双方坟场与放逐，点开可查看完整列表并移动其中的牌。
14. “释放 Token”可以自定义名称和攻防；取消“有攻防”即可创建无攻防 Token。浏览器会记忆上一次配置，下次可直接释放。

战场不会自动判断哪张牌是地。你可以：

- 把牌拖到“地”区域，它就会作为地显示。
- 把牌拖到“非地”区域，它就会作为非地显示。
牌面不会显示类型文字。内部只根据你拖到“地”还是“非地”区域来摆放。

## 当前边界

当前版本只做娱乐模拟，不做：

- 卡牌图片和官方 Oracle 文本
- 摩登禁牌表和套牌合法性校验
- 完整自动规则引擎
- 账号、天梯、匹配、回放
- 服务器持久化
